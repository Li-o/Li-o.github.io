html&css复习：第十五篇

2019-09-22

1.表单简介
用来向服务器提供请求，比如百度的搜索框注册等。
使用form来创建一个表单，form标签中必须给定一个action属性，该属性指向的是一个服务器地址，当我们提交表单时将会提交到action属性对应的地址。
使用form创建的仅仅是一个空白的表单，我们还需要向form中添加不同的表单项。
文本框：< input type = “text”/>
提交按钮：< input type = “submit” value = “提交”/>
密码框：< input type = “password”/>
如果希望表单项中的内容会提交到服务器中，还必须给表单指定一个name属性
< input type = “text” name = “username”/>
用户填写的信息会附在URL地址后边以查询字符串中的形式发生给服务器URL地址？查询字符串格式：属性名=属性值&属性名=属性值
2.表单项一
单选按钮：< input type = “radio” name = “” value = “” />
多选框：< input type = “radio” name = “” value = “” />
通过name属性来进行分组。
需要用户选择，但不需要直接填写内容的表单项，还必须指定一个value属性属性，这样被选中的表单项的value值将会最终提交给服务器。
下拉列表：使用select来创建一个下拉列表，在下拉列表中使用option标签来创建一个一个列表项，下拉列表的name属性需要指定给select，而value属性需要指定给option。
如果希望单选按钮或多选框中指定默认选中的选项，则可以添加属性checke=“checked”属性。
可以通过option中添加selected = “selected”来设置默认选项，如果给下拉列表select添加一个multiple= “multiple”则下拉列表变为一个多选的下拉列表。在select中可以使用optgroup对选项进行分组，同一个optgroup中的选项是一组，optgroup有一个属性label来指定分组名字
