2.外部样式表
外部css文件，通过link标签将外部css文件引入到当前页面
< link rel=“stylesheet” type=“text/css” href=“路径” />
将css样式统一编写到外部样式表中，完全使结构和表现分离，可以使样式在不同的页面中使用，通过link标签引入，可以利用浏览器的缓存加快用户访问速度提高用户体验。在开发中推荐使用外部css文件。

通过选择器可以选中指定元素，将声明快的样式应用在对应的元素上。
4.内联和块元素

div标签 块元素：独占一行的元素，无论内容多少都会独占一行。div标签没有任何语义，就是一个纯粹的块元素，并且不会为它里面的元素设置任何的默认样式，div元素主要用来对页面进行布局。
span标签 内联元素：只占自身大小的元素，不会占用一行。span也没有任何语义，span专门用来选中文字，为文字设置样式。
块元素主要用来做页面布局，内联元素主要用来选中文字设置样式，一般情况下只使用块元素去包含内联元素，不会去使用内联元素包含块元素。
a标签可以包含任意元素
p标签不可以包含任意其他块元素
5.常用选择器

元素选择器：通过元素选择器选择页面中所有指定元素。
语法：标签名{ }
id选择器：通过元素id属性值选中唯一的元素
语法：#id属性值{ }
类选择器：通过元素的class属性值选中一组元素
语法：.class属性值{ }
可以同时为一个元素设置多个class值，多个值之间使用空格分开。
选择器分组：通过选择器分组可以同时选择多个选择器对应的元素
语法：选择器1，选择器2，...选择器N{ }
通配选择器：可以用来选中页面中所有的元素
语法：* { }
交集选择器：也叫复合选择器，可以选中同时满足多个选择器的元素
语法：选择器1 选择器2 选择器N { }
对于id来说不建议使用交集选择器。
