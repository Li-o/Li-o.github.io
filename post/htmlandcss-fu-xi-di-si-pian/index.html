1.伪类选择器
专门用来表示元素的一种特殊的状态。
例如：访问过的超链接，普通的超链接，获取焦点的文本框
当我们需要处理在这些特殊状态元素设置样式时，就可使用伪类。
-a:link 正常连接
-a:visited 访问过连接
-a:hover 鼠标移入
-a:active 超链接被点击状态
其他元素也可以使用hover和active
注：当给a标签添加伪类的时候，要按照l v h a的顺序编写

2.伪元素
用来表示元素中的一些特殊位置
首字母元素：first-letter { }
第一行元素：first-line { }
元素最前边部分元素：before { }
元素最后边部分元素：after { }
一般before都需要结合content样式一起使用，通过content可以想before或after的位置添加内容。

3.属性选择器
可以根据元素中的属性或属性值来选取指定元素
语法：
元素【属性】{ }选取含有指定属性的元素
元素【属性=“值”】{ }选取含有属性值的元素
元素【属性^=“值”】{ }选取以指定内容开头的元素
元素【属性$=“值”】{ }选取以指定内容结尾的元素
元素【属性*=“值”】{ }选取含有特定指的元素

4.子元素的伪类
：first-child 选择第一个子标签，父标签的第一个子标签
：last-child 选择最后一个子标签
：nth-child（）{ } 可以选中任意位置子标签，可以指定一个参数，指定选中第几个子元素
even 偶数
odd 奇数
：first-of-type
：last-of-type
：nth-of-type
这三个和first-child非常相似，只不过child是在所有的子元素中排列，而type是在当前类型的子元素排列。

5.兄弟元素选择器
可以选择一个标签后紧挨着的指定的兄弟元素
语法：
前一个+后一个{ }
