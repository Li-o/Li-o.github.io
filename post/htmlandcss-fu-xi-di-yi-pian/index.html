1.html结构
< !DOCTYPE html>//文档声明
< html>//根标签
< head>//头标签
< meta charset="utf-8">//设置字符集
< title>< /title>//搜索引擎首先检索title标签
< /head>
< body>//身体标签
< /body>
< /html>

2.乱码问题
UTF-8 万国码
GB2312 国标码
编码：字符 -> 二进制
解码：二进制 -> 字符
字符集：编码和解码所用规则
在中文系统中浏览器，默认都是GB2312
设置网页的一些元数据，例字符集 关键字 简介

3.常用标签
标题标签：h1--h6 h1会影响到搜索引擎中的排名
段落标签：p
换行标签：br/
水平线：hr/
在html中再多的空格 = 一个空格
换行 = 一个空格解析

4.实体
在html中一些特殊字符不能直接使用，需要一些特殊字符号来表示，这些特殊符号我们称为实体（转义字符串）
语法：&实体的名字；
空格：&nbsp；
版权符号：&copy；

5.图片标签
< img src="图片路径" alt="图片描述" />
搜索引擎可以通过alt属性来识别不同的图片
width：设置宽度
height：设置高度
宽度和高度只设置一个，图片会等比列缩放。
