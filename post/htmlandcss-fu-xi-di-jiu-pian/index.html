盒模型-内边距
盒模型-外边距
垂直外边距的重叠
浏览器默认样式
内联元素的盒模型

1.盒模型-内边距
指的是盒子内容区与盒子边框之间的距离，一共有四个方向的内边距：
padding-top：
padding-right：
padding-bottom：
padding-left：
设置四个方向的内边距，内边距回影响盒子的可见框的大小，元素的背景会延伸到内边距
盒子可见框的宽度=左边框+左边距+内容框+右边距+右边框 高度同理
使用padding可以同时设置四个边框的样式，规则和border-width一样

2.盒模型-外边距
指的是盒子与其他盒子之间的距离，不会影响可见框大小，而是会影响到盒子位置
盒子有四个方向外边距：
margin-top：
margin-right：
margin-bottom：
margin-left：
由于页面中的元素靠左考上摆放的，所以注意当我们设置上和左外边距时，会导致盒子自身位置发生改变，设置右和下外边距会改变其他盒子位置。
外边距也可以指定一个负值，如果外边距设置是一个负值，则元素会向反方向移动。
margin还可以设置为auto，auto一般只设置给水平方向的margin，如果只指定左/右外边距margin为auto则会将外边距设置为最大值，如果将left和right同时设置为auto，则会将两侧的外边距设置相同值，就可以使元素自动在父元素中居中，所以我们经常将左右外边距设置为auto以使子元素在父元素中水平居中。
外边距同样可以使用简写属性margin，同时设置四个外边距规则和padding一样

3.垂直外边距的重叠
在网页中垂直方向相邻外边距会发生外边距重叠，所谓的外边距重叠指兄弟元素之间的相邻外边距会取最大值而不是取和。
如果父子元素垂直外边距相邻了，则子元素的外边距会设置给父元素。

4.浏览器默认样式
body： {
margin ： 0 ；
padding ： 0 ；
}
可以用 * 代替body

5.内联元素的盒模型
内联元素不能设置width和height
内联元素可以设置水平方向内边距
内联元素可以设置垂直方向内边距，但是不会影响布局
内联元素可以设置边框，但是垂直方向边框不会影响布局
内联元素支持水平方向外边距，水平方向外边距不会重叠，而是求和
内联元素不支持垂直方向外边距
